Задачи парсинга:

- получить страницу с сайта  
- отфильтровать нужные данные  
- получить ссылки на файлы по фильтру
- скачать файлы в указанную папку  

*Способы реализации:*

1. вручную - циклом и вырезанием подстрок  
2. регулярками  
3. специальными библиотеками
